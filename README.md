# System Monitoring Script (Python, Linux)

A lightweight Python-based system monitoring script designed to track CPU and memory usage in a Linux environment. Built to demonstrate basic observability and reliability concepts relevant to Site Reliability Engineering (SRE).

---

## Features

- Tracks CPU usage percentage
- Monitors memory utilization
- Logs structured system metrics
- Displays real-time system statistics
- Simulates basic reliability monitoring workflow

---

## Technologies Used

- Python 3
- psutil
- logging module
- Linux environment
